# Projet_Visualisation

Ce projet a été réalisé par Stéfan BAUDIER, Arnaud FRERING et Thomas RODRIGUEZ.

Lien vers le site hébergeant la version html du Rmd :
https://arnaudfrering.github.io/Projet_Visualisation/

Nous nous sommes intéressés aux performance en fantasy league de différents joueurs de basket-ball en NBA.
Les perfomances sotn calculées d'après la formule : 
Point + Rebonds + Passes + Contres + Interceptions + Tirs marqués - Tirs ratés + Tirs à 3pts marqués - Tirs à 3pts ratés + Lancers francs marqués - Lancers francs ratés

Les données ont été recueillies par un membre du groupe (Arnaud), afin de les utiliser comme aide de prise à la décision pour jouer à une fantasy league.
Deux différents csv ont été utilisés : 
- L'un contenant un récapitulatif des perfomances d'un joueur ainsi que l'équipe pour laquelle il joue. (Tableau_Points_Fantasy.csv)
- L'autre contenant les performances à proprement parler des joueurs pour leurs différents matchs. (Donnees_Boxplot)
